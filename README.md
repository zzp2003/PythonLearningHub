## 基础算法

### 排序算法
在`sorting_algorithms.py`中实现了三种基本的排序算法：

1. **冒泡排序（Bubble Sort）**
   - 原理：通过重复遍历要排序的序列，比较相邻元素并交换它们的位置
   - 时间复杂度：O(n²)

2. **选择排序（Selection Sort）**
   - 原理：每次从未排序的序列中找到最小元素，放到已排序序列的末尾
   - 时间复杂度：O(n²)

3. **插入排序（Insertion Sort）**
   - 原理：将一个元素插入到已经排好序的序列中的适当位置
   - 时间复杂度：O(n²)
